# Isthatmelon Project

A modern web application built with React, Vite, and TypeScript.

## Getting Started

### Prerequisites

Ensure you have Node.js and npm installed. We recommend using [nvm](https://github.com/nvm-sh/nvm) to manage your Node versions.

### Local Development

Follow these steps to set up and run the project locally:

```sh
# Step 1: Clone the repository
git clone https://github.com/IsThatMelon/isthatmelon.git

# Step 2: Navigate to the project directory
cd isthatmelon

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the local development server
npm run dev
